# Next.js Job Portal

A job portal web application built using **Next.js (App Router)** and **Tailwind CSS**.

## Tech Stack
- Next.js (App Router)
- JavaScript
- Tailwind CSS
- API Routes (if planned)
- Vercel (for deployment)

## Features (WIP)
- Job listings
- Job details page
- Search & filters
- Responsive UI
- Authentication (planned)

## Branch Strategy
- `dev` – active development
- `staging` – pre-production testing
- `main` – production-ready code

## Getting Started

```bash
npm install
npm run dev
